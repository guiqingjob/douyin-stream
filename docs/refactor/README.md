## 如何使用

1. **规划阶段**: 阅读 `01-overview.md` 了解重构目标和范围
2. **设计阶段**: 阅读 `02-strategy.md` 了解技术方案
3. **执行阶段**: 参考 `03-implementation.md` 按计划执行
4. **质量保障**: 遵循 `04-quality.md` 的测试策略
5. **风险控制**: 参考 `05-risk.md` 识别和应对风险
6. **验收阶段**: 使用 `06-acceptance.md` 验证重构成果
