### 7.4 指标计算公式

#### 7.4.1 圈复杂度

```
圈复杂度 = 1 + 决策点数量
决策点 = if + for + while + and + or + case
```

#### 7.4.2 测试覆盖率

```
测试覆盖率 = (被测试覆盖的代码行数 / 总代码行数) × 100%
```

#### 7.4.3 代码重复率

```
代码重复率 = (重复代码行数 / 总代码行数) × 100%
```
