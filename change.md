# 功能
全场景下载支持：
单链接提取：输入单一视频链接或博主主页链接，极速下载。
全量/增量下载：对指定博主主页进行全量抓取；自动对比本地 SQLite 数据库（按 aweme_id），实现智能增量下载，跳过已存视频。
批量队列下载：一次性指定多个博主（从关注列表读取），后台自动排队批量处理。
本地资产整理：
博主专属目录：自动提取博主昵称作为文件夹名（如 Downloads/罗翔说刑法/xxx.mp4），极速定位素材。
自定义路径：支持脱离默认的 Downloads 目录，自由挂载外置硬盘。
空间优化：内置 ffmpeg 视频压缩模块，大幅降低囤积素材的存储压力。
关注列表管理：通过 following.json 和本地数据库双向维护，形成固定的素材“采集团队”。

智能转写引擎

自动化批量转录：自动提取视频中的音频并上传至 Qwen，支持多任务队列，单次最高可处理 50 个文件。
结构化文稿导出：
转写完成后，自动导出为 Markdown 格式。
目录结构隔离：强制遵循 转写文稿库/博主名称/视频标题.md 的层级结构，与视频素材库形成完美映射。
云端与本地空间释放：
阅后即焚：一旦转写内容成功导出到本地，系统会自动调用接口删除云端（Qwen 空间）上的暂存音视频文件，保护隐私并释放云端容量。
多账户与配额调度：
自动签到：针对每个账户，自动执行签到动作，领取免费使用时长。
状态看板：实时查询剩余转写额度、已用时长。
(设计中) 多账户轮询：当 A 账户额度耗尽时，队列自动无缝切换至 B 账户继续转写。

Web 控制台
核心工作流可视化：左侧操作表单 + 右侧全局悬浮任务队列，支持在 Web 上直接粘贴链接发起下载或转写，进度实时无闪烁更新。
全局资产大盘：在一个页面内同时管理“视频素材库”和“转写文稿库”，并提供“发现视频未转写 -> 一键发起转写”的跨模块联动按钮。
可视化系统配置：在界面上直接进行环境自检、账号 Cookie 的手动录入与配额查询。
