# 🌟 抖音短视频自动下载管家 —— 零基础使用说明书

欢迎使用“抖音短视频自动下载管家”！这是一款专门为短视频创作者、运营人员和自媒体小白设计的全自动视频搬运工具。

你只需要把喜欢的博主主页链接告诉它，它就能自动把该博主所有的无水印高清视频、点赞量、评论量等数据，整整齐齐地下载到你的电脑里，并且自动生成一个非常漂亮的网页供你随时浏览和搜索。

---

## 📖 第一部分：它能为你做什么？

- **🎬 自动下载无水印视频**：无论是单个视频，还是某个博主的所有历史视频，它都能原画质下载。
- **🧠 聪明地“只下载新视频”**：今天下载了 100 个，明天博主更新了 1 个。它第二次运行会自动跳过那 100 个旧的，只下载这 1 个新的，绝不浪费时间！
- **📊 自动记录点赞和评论数据**：视频火不火？它会把播放量、点赞数、发布时间自动存下来，方便你做竞品分析。
- **🌐 自动生成精美网页看板**：下载完后，双击打开一个网页，就能像逛精美的私人短视频库一样，随时搜索和播放。

## 💻 第二部分：你需要准备什么？（环境要求）

这个工具不需要昂贵的电脑，普通的办公本就能跑，但你需要先安装几个基础软件。

### 1. 硬件要求
- **系统**：Windows 10/11，或 苹果 macOS，或普通的 Linux 服务器都可以。
- **硬盘**：看你要下载多少视频，建议电脑至少留出 10GB 以上的剩余空间。

### 2. 必须安装的软件：Python（核心运行环境）
这个工具是基于 Python 语言写的，所以你必须先在电脑上安装 Python。
- **要求版本**：Python 3.9 到 3.13 之间（太旧或太新都不行）。
- **去哪下载**：打开浏览器，访问 [Python 官方网站](https://www.python.org/downloads/)，点击黄色的 "Download Python" 按钮，下载后像安装普通软件一样疯狂点“下一步”即可。
  > **⚠️ 新手绝对要注意的坑**：在 Windows 安装 Python 时，第一步界面底部有一个 **`Add Python to PATH`**（将 Python 添加到系统环境变量）的复选框，**一定要打勾！一定要打勾！**

## 🚀 第三部分：手把手教你安装

现在，我们把项目代码下载到了电脑里（假设放在了 `D:\douyindownload` 文件夹中）。接下来我们要给它安装“发动机”。

### 第 1 步：打开命令黑框框
- **Windows 用户**：按下键盘上的 `Win + R`，输入 `cmd`，然后按回车，会弹出一个黑色的命令提示符窗口。
- **苹果 Mac 用户**：在启动台中找到并打开“终端 (Terminal)”应用。

### 第 2 步：进入项目文件夹
在黑框框里输入以下命令（注意把路径换成你实际存放代码的地方），然后按回车：
```bash
cd D:\douyindownload
```

### 第 3 步：一键安装所需零件
继续在黑框框里复制粘贴这行代码，按回车，等待它自动下载完毕（需要保持网络畅通）：
```bash
pip install f2 playwright pyyaml httpx aiofiles
```
装完上面那个后，再运行这个命令（这个是用来安装自动弹出的浏览器的）：
```bash
playwright install chromium
```

### 第 4 步：检查是否全部搞定
我们在项目里准备了一个“体检医生”，输入下面这行命令：
```bash
python scripts/check_env.py
```
如果屏幕上打印出 `✓ 环境检测通过，可以正常使用!`，恭喜你，最难的一步已经熬过去了！🎉

## 🎮 第四部分：开始使用（具体操作流程）

工具装好了，我们来实际下载第一个博主的视频吧！

### 第 1 步：获取你的“抖音通行证” (扫码登录)
为了能看到高清无水印的视频，工具需要借用你的抖音账号登录一下。别担心，这绝对安全。

在黑框框输入：
```bash
python scripts/login.py --persist
```
**会发生什么？**
1. 电脑会自动弹出一个类似于 Chrome 的浏览器窗口。
2. 页面上会显示一个抖音的登录二维码。
3. 掏出你的手机，打开抖音 App，扫一扫屏幕上的二维码并确认登录。
4. 登录成功后，黑框框会提示 `✅ Cookie 格式及有效性验证通过`，然后浏览器会自动关闭。这就代表工具已经拿到你的“通行证”了！

### 第 2 步：告诉它你想下载谁 (添加关注名单)
去电脑或手机上的抖音，找到你喜欢的博主的主页，点击分享，复制主页链接（比如：`https://www.douyin.com/user/MS4w...`）。

在黑框框输入下面这行命令（注意把链接换成你复制的）：
```bash
python scripts/manage-following.py --add "你的抖音博主主页链接"
```
**会发生什么？**
它会提示 `✅ 已添加用户: 博主名字`。这意味着这个博主已经被加入了你的“VIP 自动下载白名单”中了。

### 第 3 步：一键开始全自动下载
现在，让工具开始干活吧！
```bash
python scripts/batch-download.py
```
**会发生什么？**
1. 屏幕上会弹出一个菜单，问你是要下载所有人，还是只下载刚才添加的某个人。你可以按提示输入数字。
2. 接着，你会看到屏幕上一行行闪过 `[完成]：视频名字...`，这意味着视频正在飞速下载到你的电脑里！
3. 下载好的视频，会自动存放在你项目文件夹里的 `downloads/博主名字` 目录下。

### 第 4 步：在精美网页中欣赏你的战利品
下载完几百个视频后，一个个去文件夹里点开看太麻烦了。
在黑框框输入最后一行命令：
```bash
python scripts/generate-data.py
```
**会发生什么？**
它会把所有视频和点赞数据整理好。你只需要打开电脑的文件管理器，找到项目里的 `downloads` 文件夹，**双击里面的 `index.html` 文件**。
你的浏览器会打开一个极具科技感、超漂亮的面板！在里面你可以：
- 搜索视频标题。
- 按点赞量从高到低排序，一眼看出哪个视频最火。
- 直接在网页里点击播放。

## ❓ 第五部分：常见问题与急救指南 (FAQ)

**Q1：输入 `python` 提示“不是内部或外部命令”？**
> **原因**：你在安装 Python 时，忘记勾选 `Add Python to PATH` 那个选项了。
> **急救**：卸载 Python 重新安装一次，千万记得在第一步勾选最下面的框。

**Q2：下载视频时提示 `⚠️ 提取到的 Cookie 验证未通过` 或一直下载失败？**
> **原因**：你的抖音登录状态过期了，或者你在网页上退出了登录。
> **急救**：重新在黑框框里运行 `python scripts/login.py --persist`，重新用手机扫一次码就好了。

**Q3：弹不出扫码的浏览器，提示 `Playwright 未安装`？**
> **原因**：第三部分安装环境时，你漏掉了最后一步。
> **急救**：在黑框框里运行 `playwright install chromium`，等它下载完。

**Q4：为什么有些视频没被下载下来？**
> **原因**：这个工具非常聪明，默认开启了**“增量更新”**。只要你电脑里已经有这个视频了，它就会自动跳过，绝不重复下载浪费你的硬盘。

**Q5：网页里的视频封面怎么是黑的，或者数据不对？**
> **原因**：你可能下载了新视频，但是忘了更新网页数据。
> **急救**：每次下载完新视频后，记得运行一下 `python scripts/generate-data.py`，然后再刷新你的浏览器页面。
